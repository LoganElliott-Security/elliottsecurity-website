## Overview

Investigations lose momentum when analysts repeatedly search for the same event ID meanings and field definitions. This cheat sheet consolidates high-value Windows and Sysmon events into a single printable reference.

## What's Included

- Critical authentication and logon events
- Common Sysmon IDs for process, network, and registry activity
- Quick triage questions for each event category
- MITRE ATT&CK mapping references for common patterns

## Intended Audience

- Tier 1 and Tier 2 SOC analysts
- Incident responders performing initial scoping
- Detection engineers validating new rule hypotheses

## Maintenance

The guide is versioned in Git and updated as detection priorities evolve. Community feedback and real incident lessons drive each revision.
