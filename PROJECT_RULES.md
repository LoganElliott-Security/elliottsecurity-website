# ElliottSecurity Platform Rules

## Philosophy

Markdown is the CMS.

GitHub is the source of truth.

Cloudflare Pages deploys automatically.

The website is a rendering engine.

---

## Folder Structure

Use lowercase directories only.

components/

layouts/

content/

config/

styles/

types/

---

## Code Style

TypeScript everywhere.

Astro best practices.

Reusable components.

No duplicated code.

Responsive by default.

Accessibility first.

---

## Content

Everything must use Astro Content Collections.

Never hardcode projects.

Never hardcode articles.

Never hardcode threat hunts.

All content comes from Markdown.

---

## Git

Small commits.

Semantic commit messages.

Never break the build.
