# Hunt Methodology

> [!summary] Summary
> Standard threat hunting loop for ElliottSecurity.

1. Intelligence / hypothesis
2. Scope and data availability check
3. Query / collect
4. Analyze
5. Document findings
6. Convert outcomes into detections or IR improvements
