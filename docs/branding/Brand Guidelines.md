# ElliottSecurity Brand Guidelines

Version: 1.0  
Platform: ElliottSecurity Platform

---

## Purpose

ElliottSecurity is a professional cybersecurity engineering platform.

Branding must communicate:

- Precision
- Trust
- Engineering discipline
- Enterprise readiness

The brand should feel at home beside Elastic, CrowdStrike, SentinelOne, Palo Alto Networks, Microsoft Security, and Cloudflare.

---

## Brand Mark

The primary brand mark is a **custom geometric monogram**:

- A constructed letter **E** (not a typeface glyph)
- Enclosed in a subtle **hexagonal shield**
- Rendered with a restrained purple gradient from design tokens

The monogram is the primary icon for:

- Favicon
- Navigation
- Footer
- PWA / mobile home screen
- Social avatar
- Loading indicator
- Open Graph branding

Source files:

| Asset | Path |

## Personality

| Do | Don't |
|----|-------|
| Minimal | Cyberpunk |
| Premium | Hacker clichés |
| Geometric | Flames, skulls, locks |
| Technical | Circuit-board ornament |
| Elegant | Neon glow excess |

---

## Voice (visual)

- Dark surfaces by default (`#09090B`)
- Purple accents for signal and hierarchy
- Large spacing, quiet confidence
- Motion used sparingly for presence, not decoration
